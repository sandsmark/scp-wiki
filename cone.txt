Special Containment Procedures
[[/div]]

The original 8x8 font provided by the BIOS on the IBM PC and family (PCjr, XT, AT) is normally used in graphics modes.  Text modes use the fonts provided by the character ROM (or BIOS ROM) on the video card; see below for these.  The BIOS font contains only the lower 128 ASCII characters, but the same 8x8 font was carried over to IBM's EGA and VGA, which combine it with the upper half of the CGA charset.  This fills out the CP437 character set used here.
