A method of amnesticating only the false memories has not yet been found.</p><p><b>Addendum 2MIN-A:</b> Experiment Logs 2MIN-B and 2MIN-D</p><p><div class='collapsible-block 2min-b'><div class='collapsible-block-folded' style='display: block;'><a class='collapsible-block-link' target='2min-b'>► Experiment Log 2MIN-01</a></div><div class='collapsible-block-unfolded' style='display: none;'><div class='collapsible-block-unfolded-link'><a class='collapsible-block-link' target='2min-b'>▼ Experiment Log 2MIN-01</a></div><div class='collapsible-block-content' style='display: block;'><blockquote><p style='text-align: center;'><b>EXPERIMENT LOG 2MIN-01</b></p><hr><p>The purpose of this experiment was to establish a firsthand written description of SCP-2MIN and then compare this description with another observer.</p><p>D-68134 was given a pencil, clipboard and a single sheet of paper. I don't understand why I have to write about an empty room.</p></blockquote></div></div></div></p><p><b>Addendum 2MIN-B:</b> Empirical data</p><p>As an attempt to determine the true nature of SCP-2MIN, a collection of data has been recorded from SCP-2MIN and is listed below. I, uh... might have majorly fucked up by doing this.</p><p>It's really weird seeing all the things you can't usually remember. There's bugs everywhere, covering every surface.
