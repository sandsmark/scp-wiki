[[collapsible show="▷ Fifth Precept" hide="▽ You can make custom quoteblocks!"]]

------

Okay, this one is genuinely really cool to me. So, you //could// adjust every quoteblock on a page using CSS, or you can adjust just a few or only one while leaving the rest untouched. Try it with the code below!

[[code]]
[[module css]]
.nagiros blockquote{
[argument];
}
[[/module]]
[[/code]]

And then where you want the quoteblock:

[[code]]
[[div class="nagiros"]]
> This will reflect your special arguments.
[[/div]]
[[/code]]
